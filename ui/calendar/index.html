参数设置

参数 默认值 类型 描述
container '' String

日历生成容器，HTML DOM 选择器
triggerNode '' String

开始触发HTML DOM 选择器
endTriggerNode '' String

二次触发HTML DOM 选择器
count 2 Number 显示月份数
stepMonth 1 Number 上一月，下一月切换步进
afterDays -1 Number

酒店日历，-1为没有限制，其他依次类推
weekPrefix '星期' String 日历星期 前序，一般为周 或 星期
isHoliday true String 设定节假日
firstDayOfWeek 0 String 0表示星期天，以此类推
classNames:{}   Object ------------------日期相关className-----------------
classNames.oldDate 'oldDate' String,[Bloolean] 过期日期className，同时具有Boolean 判断
classNames.overDate
'overDate' String,[Bloolean] 超出范围日期className，同时具有Boolean 判断
classNames.weekend
'weekend' String 周末相关className
classNames.hoverRangeDay
'hover' String,[Bloolean] 滑过选定日期className，同时具有Boolean 判断
classNames.selectRangeDay 'selected' String,[Bloolean] 选定日期className，同时具有Boolean 判断
textNames:{}   Object ------------------日历相关字段文本---------------------
textNames.monthNames '' Array,String 月份名称，可选值：“” || 数组
textNames.dayNames ['日', '一', '二', '三', '四', '五', '六'] Array 月份名称，可选值：“” || 数组
changeYear false Boolean show select year
changeMonth false Boolean show select year
rangeYear [1900,new Date().getFullYear()] Array range of the changeYear
!rangeLock false Boolean 设定日历在超出范围的时候，月份是否可切换
