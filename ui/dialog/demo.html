参数设置:回到顶部↑

参数 默认值 类型 描述
新type undefined String

对话框类型，如，['成功'，‘失败’,‘异常’]，不同类型模版之间差异于图标，模版绑定类型，

如： <span class="dialog_type_{{type}}"></span>
title null String

对话框标题，

模版绑定对话框标题：..</div><span class="title">{{title}}</span><..
content null String

对话框内容，

模版绑定对话框内容：..</div><span class="title">{{content}}</span><..
timeout 0 Number 适用于['发表成功'，‘网络延时’，'加载中']等提示，但是注意加载中异步问题，参考腾讯提示-异常提示
zIndex 10001 Number 对话框层高度，mask设置为 true， mask（阴影层自动 -1）
mask false Boolean

阴影层，依赖 $.mask,$.unmask
opacity 0.5 Number 阴影层透明度，[0,1]
background "#000" String 阴影层颜色
button null Array 
			button:[{
					caption: "句柄字段",
					name:"这里是名字",
					callback: function() {
                      // 按钮点击回调函数
					}
				},
				{
					caption: "句柄字段",
					name:"这里是名字",
					callback: function() {
                      // 按钮点击回调函数
					}
				}		
			]
close null Function close按钮执行事件，相当于button 中的callback: function() {},查看按钮详细说明
ok null Function ok按钮执行事，相当于button 中的callback: function() {},查看按钮详细说明
cancel null Function cancel按钮执行事件，相当于button 中的callback: function() {},查看按钮详细说明
okVal null String ok按钮的名称，相当于button 中的name:"这里是名字",查看按钮详细说明
cancelVal null String cancel按钮的名称，相当于button 中的name:"这里是名字",查看按钮详细说明
before function(){} Function 打开对话框 前调函数
callback function(){} Function 关闭对话框 回调函数
新showCallback null Function 对话框显示后回调函数，用于比较复杂的对话框，如：绑定事件

按钮详细说明:回到顶部↑

			if(op.close) { //常用按钮 关闭
				//data.close = "关闭";
				op.button = op.button || [];
				op.button[op.button.length] = {
					caption: "close",
					name: "关闭",
					callback: isFunction(op.close) ? op.close : function() {
						fn.hide();
					}
				}
			}
			if(op.ok) { //常用按钮 OK
				//data.ok = this.okVal;
				op.button = op.button || [];
				op.button[op.button.length] = {
					caption: "ok",
					name: this.okVal || "确定",
					callback: isFunction(op.ok) ? op.ok : function() {
						fn.hide();
						op.callback && op.callback();
					}
				}
			}
			if(op.cancel) { //常用按钮 cancel
				//data.cancel = cancelVal;
				op.button = op.button || [];
				op.button[op.button.length] = {
					caption: "cancel",
					name: this.cancelVal || "取消",
					callback: isFunction(op.cancel) ? op.cancel : function() {
						fn.hide();
						op.callback && op.callback();
					}
				}

			}
			if(op.button) { // 自定义按钮
				for(var i = 0; i < op.button.length; i++) {
					data[op.button[i]['caption']] = op.button[i]['name'];
				};
			}
			this.template = this._template.format(data);

样式约定：回到顶部↑

data-dialog-btn 属性 用于绑定事件， close,ok,cancel 是常用按钮. 以 ‘{{close}}，{{ok}}’等 是 模板字符串.
