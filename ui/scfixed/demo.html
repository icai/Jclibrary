jQuery scrollFixed || itour.util.scrollFixed demo 说明

scrollFixed: 顾名思义，页面滚动时固定某个元素

示例代码


        $('#demo1').scrollFixed({top:20});
        $('#demo2').scrollFixed({top:60});

默认参数：


        $.fn.scrollFixed.defaults = {
          top:0,
          zIndex:300,
          limit:null
        };

*IE6 防止闪屏：

        *html{
            background-image:url(about:blank);
            background-attachment:fixed;
        }

注意事项及说明：

limit 属性 ：限制滚动不能溢出的页面高度，一般是父节点，或者父节点的同级元素的最高高度的元素 (String {jquery selector} )。

top 属性 ：元素停靠屏幕时，距离屏幕顶部的高度。

itour 命名空间支持：


    function __import(o,param){
      $(o).scrollFixed(param)
    }

    exports.itour = exports.itour ? exports.itour : {};
    exports.itour.util = exports.itour.util ? exports.itour.util : {};
    exports.itour.util.scrollFixed = __import;
