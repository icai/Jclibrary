回到顶部↑类变量/函数说明:

变量/函数 默认值 类型 描述
addClass 'current' String

tab触发事件时，添加的className

trigger 'click' String

触发事件，可选值：'mouseover'
handleBox '.Jc_handlebox' String[selector] handle Box
handleClass '.Jc_handle' String[selector]

handle Class
panelBox '.Jc_panelbox' String[selector]

panel Box
panelClass '.Jc_panel' String[selector] panelClass
callback null Function 回调函数
content null Function 动态设置::func

		index: 0,
		handleClass:null,

		panelClass:null,
	});
