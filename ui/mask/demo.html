jQuery mask || itour.ui.mask demo 说明

demo：

  • 0.默认状态（打开）

  • 
				$.mask();

  • 1.默认状态（关闭）

  • 				$.unmask();

  • 2.设置样式参数

  • 				$.mask({
					background:"#2932E1",
					zIndex:9999,
					opacity:0.7
				});

  • 3.直接删除:

  • 				$.unmask("fast");

  • 4.删除动画过渡时间:

  • 				$.unmask(2000);

  • 5.删除所有遮罩层:

  • 				$.unmask(true,2000); // 第一个参数设置成 true

  • 6.mask 函数 before,callback :

  • 				
				$.mask({
					background:"#FEB20F",
					zIndex:9999,
					opacity:0.8,
					before:function(r){
						alert("将会生成的遮罩层id:" + r);
					},
					callback:function(r){
						alert("已经生成的遮罩层id:" + r);
					}
				});

  • 7.unmask 函数 (待续) :

  • 				
				$.mask(function(){

				});
